# Design System

## Brand Identity
- **Keywords**: Trust, Professionalism, Calm, Premium.
- **Palette**: Deep Navy (Trust), Soft Gold/Bronze (Premium), Clean White/Gray (Modernity).

## Typography
- **Headings**: Serif font (e.g., *Inter* or *Playfair Display*) for authority.
- **Body**: Sans-serif (e.g., *Inter* or *Roboto*) for readability.

## Components
### Buttons
- **Primary**: Solid Navy, rounded corners.
- **Secondary**: Outline or soft background.
- **CTA**: Distinct color (e.g., Gold accent).

### Cards
- Subtle shadow (`shadow-sm` to `shadow-md`), white background, rounded-lg.

## Accessibility
- WCAG AA standard.
- High contrast text.
- Focus rings visible on all interactive elements.
